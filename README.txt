LifeRay Sales Tax Challenge

This application takes the input given from the challenge, and produces the specified output. It instantiates multiple Item classes and inserts them into an ArrayList inside a Cart class. It then performs some tax calculations on the items within the carts to obtain and display totals.


Prerequisites
Java installed on your device


How to build
The application can be built by running "Build_Jar.bat" located in the project's root directory


How to run
Run the executable .jar entitled "Sales_Tax_Challenge" located in build/libs

If for some reason your pc doesn't execute it, navigate to the directory and enter the command: java -jar "Sales_Tax_Challenge.jar"
